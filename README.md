- Vite.js
- React

- Tailwind CSS
- Responsividade
- Typescript
- Jest

## Instruções
- Clone o repositório
- Instale as dependências `npm i || yarn`
- Rode o app em modo de desenvolvimento `npm run dev || yarn dev`
